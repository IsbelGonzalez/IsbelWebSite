Conteiner propierties

container {display: }

Se agrega al contenedor o caja y permite desplegar las propiedades anidadas a esta.

  • block default..
  • flex Caja Flexible.

container {flex-direction: }

Indica la direccion a la cual la caja se hace flexible.

Establece el eje al cual la caja se hace flexible. main axis X , cross axis Y

  • row default:Se flexiona de forma horizontal.
  • column Se despliega en forma vertical.
  • row-reverse Se flexiona de forma horizontal y los Item se revierten.
  • column-reverse Se despliega en forma vertical y los item se revierte.

container {height:}

  • 50vh El Item llenara el 50 % del la vista disponible view port heigth
  • 100vhEl Item llena el 100% de la vista disponible .

container {flex-wrap: }

Permite que los Item colapcen al cumplir su tamano definido.

Es el ancho del Item que aunque se cree fijo esto depende de las otras propiedades del box.

Si el flex-grow la fila crece pero colapsa a un tamano cercano al definido solo crece para rellenar lo minimo posible.

En modo column tiene que tener el contenedor un height establecido

  • wrap Hace que los Item colapcen al siguente nivel si no caben.
  • nowrap default
  • wrap-reverse Colapsan en reversa.

container {justify-content: }

Distribuye y alinea en el eje principal que por defecto es el horisontal.

Debes tener flex-direction: row; para que el sentido sea en el eje horizontal. Y colum para el eje vertical.

  • center Alinea en el centro.
  • flex-start defaultAlinea al inicio. Es el valor por defecto.
  • flex-end Alinea al final.
  • space-around Ocupa el espacio disponible.
  • space-between Ocupa el espacio disponible pero los Item se situan desde el inicio y el final.

container {align-items: }

Distribuye y alinea en el eje opuesto al por defecto .

Crux axis

  • center Alinea en el centro del eje cruz.
  • flex-end Alinea al final del eje.
  • flex-start Alinea al inicio del eje.
  • stretch Alinea ocupando el espacio disponible.
  • baseline Se alinea es la tipografia todas las letras en su linea de base de las letras .

Items propiedades

containerItem {width: }

Es el ancho del Item que aunque se cree fijo esto depende de las otras propiedades del box.

  • 300px Si flex-grow esta activo los Item se ajustan a la pantalla pero con un tamano cercano al definido.
  • 400px Si no esta activo el flex-grow seran el width definido y los otros colapsan dependiendo del ancho de pantalla.

containerItem {order: }

Se puede cambiar el orden de los Item.

Incluso podemos agregar reglas que cambien los item dependiendo del usuario o el navegador.

Aunque visualmente el orden cambia el navegador no sabe esto la seleccion sigue igual es solo efecto visual.

  • 0 Este es el valor de todos los Item por defecto.
  • 1 Se coloca primero todos los que tienen 0 y luego 1.
  • 2 Continuo al 1.
  • -1 Tambien aceptan numeros negativos.

containerItem {align-self: }

Sobre escribe la posicion de un item en particular en el cruss axis.

  • center Aliena el Item en el centro.
  • flex-start Aliena inicio del eje.
  • flex-end Aliena fin del eje.
  • stretch Ajusta el item para que ocupe el espacio disponible.
  • baseline Alinea sobre la liena base

containerItem {flex: }

Indica lo que se hace con el espacio sobrlare.

la palabra flex es un diminutivo para varios parametros flex-grow flex-shrink flex-basis

El valor que tome es una proporcion del espacio sobrate y no el tamano del item

  • 0 1 auto Valor por defecto y toma el espcio de la caja es decir no hace nada con el espacio sobrante.

containerItem {flex-grow: } recortado flex

Es el factor de crecimiento y recive numeros positivos.

Proporcion del espacion del contenedor para repartir a los Item q estan dentro del contenedor.

Crece en el main axis.

  • 1 Es una proporcion donde todos los items temdran el mismo tamano.
  • 0.5 Un Item con la mitad del tamano del espacio sobrante.
  • 1.5 La proporcion del Item sera de 1.5 del espacio sobrante.

containerItem {flex-shrink: } recortado flex

La proporcion a la cual el item se reduce cuando el espacio se reduce.

  • 1 Valor por defecto proporcion uno
  • 2 Se reduce al doble de la velocidad

containerItem {flex-basic: }

Es el tamano ideal de la caja y lo ocupa cuando esta disponible.
