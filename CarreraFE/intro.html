Introduci'on al Desarrollo Web Front- End

Desarrollador Web Front-End

Un desarrollador Web Front End se encarga de crear codigo HTML, CSS, JS del lado del cliente este codigo lo interpreta el navegador, y como resultado debe ser una experiencia Web utilizable por el usuario.

Navegador

El navegador es el que interpreta los codigos HTML, CSS y JS

HTML HTML Esencial

Es el codigo que me permite crear la estructura de la pagina Web.

Es un lenguaje (comunicacion entre desarrollador y navegador) de marcado (etiquetado que usa etiquetas(tag)) .

CSS

E el codigo especializado para crear el estilo y el diseno de la pagina Web

JS

El cogigo que permite agregar la logica la interacion y animacion de las paginas Web.

Cliente/Servidor

Cliente

Cliente: Computador del usuario. (Front-End)

Servidor

Servidor: Computador especializado que contiene la informacion requerida por el usuario. (BackEnd)

Pagina Web

Pagina Web: wmedia.es es un alias dominio que se coloca a la direcion IP

Direccion IP

Direcion Ip: 138.87.69.12. Es algo unico que identifica un recurso en la Web.

Servicio DNS

Servicio DNS: Traduce el dominio (alias de la pagina Web) a la direccion IP. Incluso hay direciones IPs dinamicas por eso es requerido este traductor.

Request

Requuest: Peticion al servidor se requiere un recurso. Se negocia para pedir algo.

Response

Respuesta del servidor al cliente y esta compuesta por HTML + CSS + JS lo que entiende el navegador.

Base de datos

Generalmente es parte del servidor o se enlaza a el y guarda la indormacio

HTTP o HTTPS

Es el protocolo para trasmitir la informacion a travez de este protocolo se hace el intercambio siguiendo una serie de reglas estrictas de como trasmitor la informacion.

Perfiles de Desarrollador

Front-End

Client Side: Se encarga de todo lo implicado del lado del cliente. Enviar el request recivir el responce y traducirlo y mostrar la informacion al cliente.

Back-End

Server Side: Del lado del servidor, agregar la informacion en Base de Datos, Codigo para las capas de negocios y capas de Base de datos. Seguridad. ETC

Full-Stack

Engloba los dos mundos tanto en Front-End como el Back-End
