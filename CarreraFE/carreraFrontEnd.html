Data Type CSS

Todas las propiedades CSS aceptan un valor. Estos valores (Data types), pueden ser de diferentes tipos: color, longitud, url, "auto",etc.

Unidades CSS

Unidades Absolutas y Unidades Relaivas

Las absolutas tienen un valor que no cambia ni depende de nada. pixeles, cm, mm, in,

Las relativas estan basadas en otro valor. em, rem, vmin, vmax, vh .

Pixeles

Los pixeles nos dan una independencia total, ya que son considerados divice independent.

Un valor expresado en pixeles siempre aparecera con el mismo tamano tanto en un movil como en una pantalla retina o 5k.
